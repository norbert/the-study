# Negroni

## Variations

### IBA

* **30 ml** gin ( *Tanqueray No. 10* )
* **30 ml** sweet red vermouth ( *Antica Formula* )
* **30 ml** *Campari*
* Orange twist

Pour all ingredients directly into old-fashioned glass filled with ice. Stir gently. Garnish with half orange slice.
